# Machine Learning & Data Mining Lab Projects

This repository contains a collection of machine learning and data mining experiments implemented as part of academic lab work using Python.

## Topics Covered
- Data preprocessing and exploration
- Gradient Descent implementation from scratch
- Decision Tree Regression
- Support Vector Machines (SVR)
- Logistic Regression with regularization
- Ensemble learning (Random Forest, AdaBoost, XGBoost)
- Clustering (K-Means, Hierarchical, DBSCAN)
- K-Nearest Neighbors (KNN)
